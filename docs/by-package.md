# 📦 패키지 비교 매트릭스

본 페이지는 **6 통합 파일럿 패키지** 를 한눈에 비교합니다. 귀사 환경에 맞는 패키지를 5 분 안에 선정하세요.

!!! tip "선정 의사결정 순서"
    1. **§1 매트릭스 표** — 6 패키지를 행 단위로 비교 (도메인·기간·시나리오 등)
    2. **§2 의사결정 트리** — 분기 질문에 따라 패키지 자동 매칭
    3. **§3 paste-ready 블록 일람** — 각 패키지의 핵심 자산 빠른 식별

## 4. 패키지 변형 적용 — 5 변경 지점

매칭된 패키지를 귀사 환경으로 변형:

| 변경 지점 | 변경 대상 |
|---|---|
| **§1.2 [고객사] 가설 박스** | 새 [고객사] 가설 (4 대 정합 조건) |
| **§2.3 KPI 7 컬럼 표** | 정량 [수치] (기존·목표) |
| **§5.1·5.2 단계별 추진 목표 + 간트** | (다년 R&D 시) 단계 구간·연차 시점 |
| **§6.1·6.2 단계별 5 비목 예산** | 정량 [수치] (각 비목 액수) |
| **본문 [고객사]·[공정]·[수치]·[기간]·[%] 플레이스홀더** | 일괄 치환 |

→ 신규 작성 비율 약 **25~40%** 만으로 사업계획서 1 부 완성. 자세한 절차는 [Quick-Start §4 통합 파일럿 변형](guide/quickstart.md) 참조.
