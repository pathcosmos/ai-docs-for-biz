## 6. 4 대 AI 영역 → RACI 역할 매핑 (사내 R&R 결정 참고)

본 §6 은 사내 인력의 R&R 결정에 활용. 사업계획서 paste 시에는 §3 표만 사용.

| AI 영역 | PM | DS | MLE | DE | QA | 도메인 |
|---|:---:|:---:|:---:|:---:|:---:|:---:|
| 공정설계 자동화 (LLM+RAG, 핵심) | A | **R** | C | C | C | **R** |
| Mill Sheet OCR | A | **R** | C | C | C | I |
| MLOps 운영 환경 | A | C | **R** | C | C | — |
| 암묵지 추출·자산화 | A | **R** | I | C | I | **R** |

**약어 풀이**:
- **PM** = Project Manager (사업 책임자, 부장～차장급, **특급**)
- **DS** = Data Scientist (AI 모델 개발자, 차장～대리급, **고급**)
- **MLE** = Machine Learning Engineer (MLOps 엔지니어, 과장～대리급, **고급**)
- **DE** = Data Engineer (데이터 엔지니어, 과장～대리급, **중급**)
- **QA** = Quality Assurance (테스트·검증 실무자, 대리～사원급, **중급**)
- **도메인** = Domain Expert (사내 베테랑 숙련공 20+ 년, 1～2 명, **특급**)

**RACI**: R = 실행 책임 / A = 최종 설명 책임 (1 명 한정) / C = 사전 협의 / I = 결과 통보
